Présentation du projet

À partir d'un fichier texte contenant une liste d'actions ayant un coût et une rentabilité, créer un programme extrayant les combinaisons d'actions à acheter permettant le meilleur rendement pour un coût total défini.

Compétences évaluées

  • Configurer un environnement Python
  • Appliquer les bases de la programmation en Python
  • Utiliser le contrôle de version avec Git et GitHub
  • Gérer les données à l'aide du processus ETL
